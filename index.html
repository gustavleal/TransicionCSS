HTML 5

HTML 5

HTML5 é a mais recente evolução do padrão que define o HTML. O termo representa dois conceitos diferentes: É uma nova versão da linguagem HTML, com novos elementos, atributos, e comportamentos. e um conjunto maior de tecnologias que permite o desenvolvimento de aplicações e web sites mais diversos e poderosos.

CSS 3

CSS 3

CSS3 é a terceira versão principal da linguagem principal da World Wide Web, geralmente usada em combinação com o HTML5. O desenvolvimento do CSS3 começou em 2005 em contraste com outras especificações e modularização. Os módulos CSS3 são descritos em especificações separadas, como Seletores, Consultas de Mídia, Texto, Fundos, Cores, Transformações 2D e 3D, Transições, Animações e Multi-Colunas.

JavaSript

JAVASCRIPT

JavaScript é uma linguagem de programação voltada para aplicações web, de fácil execução e que pode ser rodada a partir de um navegador de internet. Hoje, ela é uma das mais populares do mundo, o que a faz ser considerada fundamental para programadores.

UI & UX

UI & UX

A interface do utilizador (UI) é uma das vertentes que é estudada, num produto, de modo a melhorar a experiência do mesmo. Esta vertente tem como foco o trabalho de design na interface usada. A experiência do utilizador (UX) preocupa-se em conciliar o que o consumidor necessita com o que a empresa oferece ou pode oferecer, no caso da possibilidade de criação de novos produtos ou serviços. Nesta área são usados vários testes com o objetivo de estudar o consumidor e a sua experiência com o produto ou serviço.

GitHub

GITHUB

GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão chamado Git. Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso.
